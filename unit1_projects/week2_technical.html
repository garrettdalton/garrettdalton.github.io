Technical Blog: Inline vs Inline blocks

To understand what the differences are between Inline and Inline-Block one must first know about the display property. The display property can be set by the programmer, but is often set by default depending on the element.

Paragraphs, like the one you are reading and the one above it, are set by default to display with block properties. Elements with their display property set to 'block' have a defineable width and height and will create a new line each time a block element appears on the page. Block 2

Elements which have the property of display set to inline do not have a defineable height and width and do not be displayed on a new line on the page. The height and width of the inline element will be set based upon the text contained within it. Inline elements will flow into one another, and therefore appear as if 'in line'. Setting an elements display to Inline-Block will give it attributes of both Block elements and Inline elements. These elements will have a defineable height and width, and will remain distinctly seperate. However, they will retain the ability to be horizontally displayed like inline elements.
