Classes in Ruby

Classes

ruby is an Object Oriented Programming (OOP) Language. It utilizes objects, which can be grouped according to class. A class is almost like a blueprint for objects. When you define a class, you are basically laying down a set of variables and methods for future objects of that class to use and inherit.

Inheritance? What's that?

Inheritance is a concept that is critical to object oriented languages. It allows for the reuse of code to maximize efficieny while programming. Inheritance sets up a hierachy of objects, with the child objects gaining the attributes of the parent object. A class is a sort of super object, which passes its variables and methods to its children. The children 'inherit' those attributes so that the programmer doesn't have to define the class atrributes each time they create a new object.
